## 扩展介绍
本扩展为 **[无名杀（noname）](https://github.com/libnoname/noname)** 武将扩展
基于《明日方舟》部分干员、《柚子社》部分角色、《ATRI》中拥有立绘角色及部分Galgame/ACG角色制作

## 核心功能
- 明日方舟专属**技力系统**，支持自定义上限与技能图标(通过修改代码、替换图片文件)
- 完整**藏品系统**，包含希望、源石锭等明日方舟特色机制
- 支持**千幻聆音换肤**与本体换肤
- 包含角色配音和自定义UI等扩展功能
- ACG角色武将，技能贴合原作设定

## 适配信息
- 最低适配无名杀版本：**1.11.0及以上**（推荐1.11.2+）

## 使用方法
- 使用万能导入法进行导入
- 重启游戏，在扩展列表中启用「宁宁起爆器」
